# AdventureWorks-Sales-Analysis
Sales and Customer Analytics using SQL Server
## Project Overview
This project analyzes sales performance with Revenue by Time, Products, Countries; and YoY;
and Customer behavior, using AdventureWorksDW.
## Tools
- SQL Server
- SQL
- GitHub
## Business Questions
- How has revenue changed over time?
- Which products generate the highest revenue?
- Who are the most valuable customers?
- Which countries contribute the most revenue?
## Key Metrics
- Revenue
- Orders
- Customer Lifetime Value
- Revenue Growth YoY
- Retention Rate & Churn Rate
- RFM
